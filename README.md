# trombonline
womp womp, amirite?

trombonline is an easy-to-use web-based trombone simulator! <br>
only intended for use with keyboard and mouse. <br>
supports 8 partials (B♭1 to B♭5) and all 7 slide positions! <br>
also include heavy yap for anyone new to the instrument ^-^ (please dont read it...)
